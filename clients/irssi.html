Using Irssi to Access the Desert Bus Chat

Irssi is a popular IRC client amongst enthusiasts, but not the easiest for new users. If you use Irssi, here are some tips to get up and running with the Desert Bus Chat quickly.

  1. First, (optionally) add some handy aliases so that /bid and /ns etc. work as they should (the first is the most important):
    • /alias bid quote bid
    • /alias ns quote ns
    • /alias login quote login
    • /alias id quote id
    • /alias bidserv quote bidserv
    • /alias bs quote bs
  2. Add a network:
    • /network add -nick YourNickname DesertBus
  3. Add the server:
    • /server add -ssl -auto -network DesertBus irc.desertbus.org 6697 your.email@example.com:secretpassword
    — use the email address and password of your donor account

    Note: -auto in this command tells Irssi to automatically connect to the Desert Bus chat on startup; if you would prefer not to do this, remove -auto

  4. Optionally, to hide joins, quits, etc. on the chat (which gets very busy during Desert Bus):
    • /ignore -channels #desertbus * JOINS PARTS QUITS NICKS
    • /ignore -channels #desertbus ChanServ MODES
  5. Save your configuration:
    • /save
  6. Connect:
    • /connect DesertBus

You should now be in the Desert Bus chat; enjoy!
